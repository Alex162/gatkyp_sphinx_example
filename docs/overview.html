Overview

This documentation is an easy guide how to sucessfully generate realistic MARVEL data products. The simulations are build from two popular main modules named PyEchelle and Pyxel. PyEchelle is a Python based fast generic spectrum simulator that is used to generate each object or calibration spectrum, whereas Pyxel is a general imaging detector simulation framework in Python that is used to efficiently add-on all necessary CCDs effects.
